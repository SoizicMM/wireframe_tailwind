# Activité – Reproduction d'un wireframe en HTML/CSS et Tailwind 

## 📋 Objectif

L'objectif de cette activité est de reproduire le wireframe fourni en utilisant :

- HTML
- Tailwind CSS
- CSS possible en partie bonus

Le but est de respecter au mieux la structure, la mise en page et le design du wireframe.

---

## 🚀 Consulter le projet

Vous avez deux possibilités :

### Option 1 – Consulter le code directement sur GitHub

Cliquez simplement sur les différents fichiers du dépôt pour consulter le code.

### Option 2 – Télécharger le projet

1. Cliquez sur le bouton **Code**.
2. Sélectionnez **Download ZIP**.
3. Décompressez le fichier ZIP.
4. Ouvrez le dossier dans **Visual Studio Code**.
